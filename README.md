# Oranje Onderwijs Intelligence

![Logo](public/logo-oranje-onderwijs-intelligence.webp)

## 🇳🇱 Netherlands Education Data Intelligence Platform

**Oranje Onderwijs Intelligence** adalah platform intelijen data pendidikan tingkat perusahaan (*enterprise-grade*) yang dirancang khusus untuk memetakan, menganalisis, dan memverifikasi program akademik di seluruh lanskap pendidikan Belanda.

Platform ini mengotomatiskan riset manual dengan menyediakan intelijen institusional yang terstruktur, dirancang untuk skalabilitas nasional, dan menyajikan wawasan ekosistem yang mendalam melalui visualisasi data modern.

---

## ✨ Fitur Utama

- **Enterprise Dashboard**: Antarmuka premium dengan desain minimalis dan elegan (European SaaS Style).
- **Smart Program Mapping**: Katalog program akademik yang terverifikasi dari berbagai institusi (TU Delft, Hanze, ROC, dll).
- **Ecosystem Insights**: Visualisasi data canggih untuk distribusi tipe pendidikan, level gelar, dan campuran bahasa.
- **Intelligent Search**: Navigasi pencarian pintar yang memudahkan pengguna menemukan program relevan secara instan.
- **Advanced Data Pipeline**: Arsitektur scraping modular yang dirancang untuk skala nasional (100+ sekolah).
- **Mobile First**: Desain responsif dengan pengalaman "Android-native feel" yang mulus di semua perangkat.

---

## 🚀 Teknologi yang Digunakan

Proyek ini dibangun menggunakan stack teknologi modern untuk performa dan skalabilitas maksimal:

- **Frontend**: [Next.js 15+](https://nextjs.org/) (App Router) - Framework React tercanggih untuk web.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - Untuk desain UI premium, glassmorphism, dan responsivitas.
- **Visualisasi Data**: [Recharts](https://recharts.org/) - Grafik interaktif (Bar, Pie, Area, Radar charts).
- **Icons**: [Lucide React](https://lucide.dev/) - Set ikon vektor yang bersih dan konsisten.
- **Fonts**: [Satoshi](https://www.fontshare.com/fonts/satoshi) & [Inter](https://rsms.me/inter/) - Tipografi kelas dunia untuk keterbacaan maksimal.
- **Data Backend**: Arsitektur data terstruktur dengan normalisasi otomatis dari berbagai sumber portal pendidikan.

## 🛠️ Instalasi Lokal

1. Clone repository:
   ```bash
   git clone https://github.com/Firmanstmik/oranje-onderwijs-intelligence.git
   ```
2. Install dependensi:
   ```bash
   npm install
   ```
3. Jalankan server pengembangan:
   ```bash
   npm run dev
   ```
4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
